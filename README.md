Учебный проект интернет-магазина пиццы.

## 🚀 Стек
* **Frontend:** React 18, TypeScript
* **State Management:** Redux Toolkit (RTK)
* **Routing:** React Router DOM v6
* **Backend-as-a-Service:** Supabase (хранение данных и получение API)
* **Build Tool:** Vite
* **Styling:** SCSS / CSS Modules

## 🛠 Функционал
- Просмотр каталога пицц с фильтрацией по категориям.
- Сортировка по цене и рейтингу.
- Поиск товаров в реальном времени.
- Полноценная корзина: добавление, удаление, изменение количества товаров, подсчет общей суммы.

## 🏁 Как запустить проект
1. Клонировать репозиторий: `git clone https://github.com/Vittu225/pizzaStore.git`
2. Установить зависимости: `npm install`
3. Запустить проект: `npm run dev`
